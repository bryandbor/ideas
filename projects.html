MyGala

Status: Active

Description
Lindsay Wildlife Museum "LWM" was hosting its annual gala and had previously relied on the use of Excel spreadsheets and manual check-in to track guest attendance at their gala and sort through purchases. This auction database have them an all-in-one system to streamline their process and to give their check-in and check-out volunteers networked information and simultaneous.
Special Features
  • Simple user interface with custom graphics to match gala branding
  • Individual user log-ins to protect private information and provide accountability
  • Guest and item list uploads to prepopulate database with multiple fields
  • Export functionality in text format preformatted to work with Excel
  • Check-in by last name and system automatically confirms e-mail and phone number
  • Check-out by bidder number or last name
  • Automatic e-mail receipts
  • Can check paid status for item pick-up by bid number
  • Can enter different types of items, including fully tax-deductible donations, auction purchases, and raffle tickets with custom or set pricing options
  • Simultaneous data entry capacity and automatic syncing for unlimited computers
  • Automatic calculation of amount due and estimated tax-deductible amount
  • Ability to enter type of payment and confirm payment status
Technologies Used
  • PHP
  • C++
  • HTML/CSS
  • SQL

Task Chat

Status: In Production

Description
This program will help remote workers to stay in touch with the demands of their managers and vice-versa. It will allow tasks to be split into sub-tasks making it much easier to track progress on a project. Also the program will allow an easy communication channel to managers and workers alike. This program will be initally launched as an iOS app, and will be later followed by a website/web-app.
Special Features
  • Matching iOS and Web-app
  • Ability to communicate with coworkers and managers about a task
  • Easy viewing of current progress percentage
  • Sub-divide a task for better organization
Technologies Used
  • iOS
  • SQL
  • React.js
  • PHP
  • Javascript
  • HTML/CSS
